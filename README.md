# Magic Numbers

## Overview

Magic Numbers is an interactive browser-based game that challenges players to think of a number while the application attempts to determine their selection through a series of simple questions.

Designed as both a puzzle and a demonstration of logical problem solving, the project explores algorithmic thinking, dynamic user interaction, and browser-based application development without revealing the underlying method to the player.

---

## Key Features

* Interactive number guessing game
* Dynamic question progression
* Instant number prediction
* Browser-based gameplay
* Simple and intuitive user interface
* Replayable game experience

---

## Technologies

* JavaScript
* HTML5
* CSS3

---

## Development Process

This project focused on creating an engaging interactive experience while implementing the underlying logic required to consistently determine a player's selected number.

Development included designing the game flow, generating dynamic question sets, processing player responses, updating the interface in real time, and creating an experience that feels more like a magic trick than a traditional guessing game.
